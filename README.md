## 📌 Core Project: GTVMOTOR
*Enterprise-grade automotive platform focused on high-concurrency and sub-second rendering.*

### 🎨 The Frontend (The Product)
- **Tech**: **Angular 18.2** · **Signals** (Computed & Effect) · **SSR (Express 5)** · **Tailwind 3.4** · **Vitest**
- **Features**: Advanced Appraisal Wizard (photo/doc upload), Real-time Rental/Buy-Sell flows, and a comprehensive Admin Dashboard.
- **SEO Mastery**: Custom `SeoService` with dynamic Open Graph and JSON-LD injection, fully compatible with server-side hydration.

### ⚙️ The Engine (Supporting Infrastructure)
- **Backend API**: Built with **NestJS 11** and **Node.js**, leveraging **Prisma** for type-safe database access.
- **Data & Auth**: **PostgreSQL 16** with persistent external volumes and **JWT/Passport** security.
- **Cloud & DevOps**: 
  - **AWS EC2** deployment with **Nginx** handling TLS (Certbot).
  - **Docker Compose** for API/DB isolation.
  - **AWS S3** integration via Presigned URLs for secure asset handling.
  - **Security**: Hardened network (22/80/443 only) with same-origin routing to eliminate CORS overhead.
